# Ancient Temple Escape

Ancient Temple Escape is a text-based RPG game written in Python.

## Features

- Room-based exploration
- Inventory system
- Multiple enemies
- Final boss battle
- Healing and counter mechanics
- Score and turn counter

## Requirements

- Python 3.10 or higher

## Run the game

```bash
py -m rpg_game
Example commands:

- go south
- get ancient_relic
- attack
- heal
- counter
- map
- inventory
- quit
